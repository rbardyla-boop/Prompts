# Prompt Engineering & Reality-Testing Frameworks  
**(The Alpha Suite + GROK APEX ecosystem — 2026 edition)**

Personal collection of structured prompting frameworks, torture-test methodologies, creative workflows, and meta-science tools — mostly built / adapted during 2025–2026 while exploring how to make LLMs (especially Grok) more truth-seeking, regime-aware, and resistant to institutional narrative laundering.

**Core philosophy running through most files:**

- Physical grounding beats official statistics  
- Regime changes break most static models → detect & gate dynamically  
- Deflate every performance claim ruthlessly (DSR, PSR, multiple-testing correction)  
- Incentives explain more divergences than incompetence  
- Curiosity should win over consensus / credentialism

## How the pieces usually fit together

Typical flow when investigating a messy real-world question (2025–2026 examples):

1. Start with curiosity / anomaly → `META_SCIENCE_INVESTIGATION` (darkness score + incentive map)  
2. Formulate clean research session → `GROK_APEX_4.4` + `VAL_FRAMEWORK` (vibe check, scope lock)  
3. Steal/remix prior art creatively → `KLEON_CREATIVE_LAYER`  
4. Need to ground claims in physical reality? → `THE ALPHA SUITE` suite  
   - Trade / tariff angle → `GEOPOL_ALPHA`  
   - Violence / shadow money angle → `CRIME_ALPHA`  
   - Both domains at once → `COMPARE_ALPHA`  
5. Keep sessions fun & sustainable → daily `<vibe_check>` + `<daily_share>`
